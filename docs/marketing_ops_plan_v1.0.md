# 黒猫タロット：AIエージェントによる利用促進プロモーションプラン (Marketing Ops Plan v1.0)

## 1. プロモーション戦略の全体構造

黒猫タロット Version 1.0 の核心は以下の通り：

### ✔ USP
「気分」に寄り添い、AIに最適化されすぎた世界から距離をとる
はみだし思考を返す黒猫パートナー
＝従来の占いでもAI相談でもない新カテゴリ

### ✔ マーケの使命
- 「使ってみたい」導線の最適化（TikTok → LP → Draw）
- SNSへの自然拡散（UGC）の誘発
- 毎日使いたくなる理由（Daily Ritual）の構築
- 初期は日本語圏を主軸にしつつ、英語圏ユーザーも獲得

### ✔ AIエージェント活用領域
- コンテンツ生成（動画・SNS・コピー）
- ターゲット分析
- アウトバウンド（口コミ誘発）
- パフォーマンスレポーティング
- 広告クリエイティブ生成

## 2. プロモーションの主要チャネル

### ① TikTok（最大流入源）
- 仕様書にある通り、黒猫タロットは Mobile First ＆ ワンタップで利用開始可
- TikTokは「占い × AI × 感情系」が相性抜群
- 1日1回の儀式性も TikTokアルゴ と相性がいい

### ② Instagram Reels
- 静かな物語トーン
- 黒猫 + 月の世界観が刺さる
- シェアカードの投稿でUGCが自然増幅

### ③ X（旧Twitter）
- ミニコピーで「黒猫インサイト」系投稿が相性◎
- 深夜帯の利用に刺さる（仕様書のキーワード “深夜3時の黒猫タロット”）

### ④ 海外ユーザー（英語版）
- 英語UIあり → そのまま広告可能
- Tarot × AI は世界共通の関心テーマ

## 4. AIエージェント別の“実行タスク設計”

### ◆ Agent A：SNSクリエイティブ生成担当
**タスク：**
- TikTok動画スクリプト生成
- Instagram静止画コピー生成
- シェアカードのコピー生成
- 黒猫インサイト風の文章100本自動生成
- 毎日の「投稿案」自動生成

**プロンプト構造：**
あなたは「黒猫タロット」のクリエイティブ担当。
静か・皮肉・温かい・はみだした言葉を生成する。
AI最適化社会への対抗としての黒猫パーソナリティを守ること。

### ◆ Agent B：分析担当（データ観測者）
**タスク：**
- TikTok → LP の遷移率レポート
- 利用完了率の日次分析
- 深夜利用の傾向分析
- UGC投稿の発生タイミング分析
- 英語圏流入の伸び検知

**プロンプト構造：**
あなたは黒猫タロットのデータアナリスト。
ユーザー行動の異常値、伸びポイント、改善機会を毎日レポートする。
短く、洞察的に、行動に落ちる形で出力。

### ◆ Agent C：UGC促進担当（コミュニティ生成）
**タスク：**
- X・Instagram・TikTokの自然投稿から黒猫関連投稿を収集
- コメント返信案の生成
- UGCとの“会話”を継続し、ファン化を促す
- 月間ベスト「黒猫インサイト投稿」の選定

**プロンプト構造：**
あなたは黒猫タロットのコミュニティマネージャー。
ユーザーの投稿に対して、黒猫らしいウィットと静けさで返信案を作る。
ユーザーを肯定しすぎず、距離感を保った優しさ。

### ◆ Agent D：広告クリエイティブ担当
（想定：成長フェーズで広告を投下する時期）
**タスク：**
- TikTok広告向けの動画案
- Meta広告向けのコピー
- LP改善案のA/Bテスト

**プロンプト構造：**
あなたは黒猫タロットの広告プランナー。
最適解を押しつけず、感情・静けさ・余白を使った広告案を提案する。

## 6. プロモーション戦略を支えるコア洞察

- ✔ 世の中は「効率・正論・最適化」で疲れている → 黒猫タロットは “はみだせる場所” を提供する
- ✔ AIの回答は最適化されすぎて人間味を失っている → 黒猫は “非最適化の象徴”
- ✔ 占いに飽きた人が求めているのは → 「共感」ではなく「静かな違和感」 → 黒猫はそれを正しく提供できる
- ✔ Mood-driven Insight（気分AI）は新しいカテゴリ → 他にはない体験のため、拡散しやすい
